Lib.MathCore (Група А4)
Ця бібліотека є математичним ядром проєкту Mini-ChatGPT. Вона реалізує критично важливі операції для роботи нейронних мереж: перетворення логітів у ймовірності, розрахунок функцій втрат, вибір найбільш імовірних токенів та випадковий семплінг.

Роль у системі
Бібліотека надає математичні примітиви, необхідні для етапу виводу (inference) та оцінки моделі:

Softmax: перетворення сирих виходів мережі у розподіл ймовірностей.

Cross-Entropy Loss: метрика для оцінки якості прогнозів моделі.

ArgMax: вибір токена з найвищою ймовірністю.

Sampling: вибір наступного токена на основі дискретного розподілу.

Архітектура та Структура Проєкту
Проєкт організований за модульним принципом для забезпечення тестованості та високої продуктивності:

1. Interfaces & Implementation (Domain/Application)
IMathOps: Основний інтерфейс, що визначає контракт для всіх математичних операцій.

MathOps: Статичний фасад для швидкого доступу до методів.

MathOpsImpl: Конкретна реалізація обчислень, що використовує ReadOnlySpan<float> для оптимізації роботи з пам'яттю.

2. Specialized Calculators (Calculators)
SoftmaxCalculator: Відповідає за обчислення Softmax. Реалізує стратегію числової стабільності (віднімання максимального значення перед експоненціюванням для запобігання переповненню float).

LossCalculator: Містить логіку обчислення крос-ентропії між логітами та цільовим класом.

3. Utilities & Sampling (Utilities/Sampling)
ScoreUtilities: Допоміжні методи для роботи з масивами оцінок (наприклад, пошук ArgMax).

ProbabilitySampler: Реалізує логіку вибору індексу на основі переданих ймовірностей та генератора випадкових чисел Random.

4. Low-level Ops (Optional)
MatrixOps, VectorOps: Опційні класи для базових операцій лінійної алгебри.

Опис Методів
Бібліотека реалізує наступні ключові методи:

float[] Softmax(ReadOnlySpan<float> logits):
Приймає логіти, повертає масив ймовірностей, сума яких дорівнює 1.0.

float CrossEntropyLoss(ReadOnlySpan<float> logits, int target):
Обчислює втрати для конкретного цільового індексу (target).

int ArgMax(ReadOnlySpan<float> scores):
Повертає індекс елемента з найбільшим значенням.

int SampleFromProbs(ReadOnlySpan<float> probs, Random rng):
Повертає випадковий індекс, зважений за ймовірностями з масиву probs.

Очікувана поведінка та Гарантії
При інтеграції з Lib.MathCore слід враховувати наступні технічні особливості:

Числова стабільність: Наша реалізація Softmax автоматично запобігає появі NaN або Infinity шляхом нормалізації вхідних даних через max(logits).

Робота з пам'яттю: Використання ReadOnlySpan<float> дозволяє уникнути зайвих алокацій при обробці великих векторів даних.

Контракт ймовірностей: Бібліотека гарантує, що сума результату Softmax завжди дорівнює 1 (з урахуванням мінімальної похибки float).

Тестованість: Усі операції доступні через інтерфейс IMathOps, що дозволяє легко підміняти реалізацію (Mock) під час юніт-тестування інших компонентів системи.

Критерії готовності (Definition of Done)
 Реалізовано стійкий Softmax.

 Unit-тести: перевірка суми ймовірностей Softmax (=1).

 Unit-тести: Cross-entropy для відомих розподілів.

 Unit-тести: Argmax та коректність роботи семплера.
